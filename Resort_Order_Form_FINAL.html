How to build this in Google Forms (free)

Option A (recommended): Use this HTML file as your actual form — open it on any phone browser, bookmark it, done. No Google account needed.

Option B: Follow these steps to recreate it in Google Forms so responses auto-save to Google Sheets.

1

Open Google Forms

Go to forms.google.com and click the + button to create a new blank form. Title it: "Resort Supply Order"

2

Question 1 – Name

Click "Add question" → choose Short answer → label it "Your Name" → toggle Required ON

3

Question 2 – Date

Add question → choose Date → label "Date of Order" → Required ON

4

Question 3 – Department

Add question → choose Dropdown → label "Department" → add options: Staff Kitchen / Guest Kitchen / Housekeeping / Maintenance / Dive / Other → Required ON

5

Question 4 – Category

Add question → choose Dropdown → label "Category" → add all 17 categories from the Products tab → Required ON

6

Question 5 – Product

Add question → choose Dropdown → label "Product" → paste ALL products from the Products tab (copy the list). Staff will scroll to find their item. Required ON

7

Question 6 – Quantity

Add question → choose Short answer → label "Quantity" → click three-dot menu → Response validation → Number → Greater than → 0 → Required ON

8

Question 7 – Unit

Add question → choose Dropdown → label "Unit" → add: kg / litre / pcs / box / pack / bottle / jar / tin / bag / roll / tray / pair / ream / galon / Other

9

Question 8 – Urgency

Add question → choose Multiple choice → label "Urgency" → options: Normal – next order / Urgent – within 3 days / Critical – ASAP

10

Question 9 – Notes

Add question → choose Paragraph → label "Notes / Special instructions" → Required OFF

11

Link to Google Sheets

Click the green Sheets icon in the Responses tab → "Create a new spreadsheet" → name it "Resort Supply Orders". All submissions now auto-populate that sheet.

12

Add buyer columns to the Sheet

Open the linked Sheet. After the last auto-filled column, manually add: Qty Bought / Actual Price (IDR) / Total Cost / Qty Received / Status

13

Share the form

Click Send → copy the link → share via WhatsApp to each department head. Tell them: "Bookmark this link. Fill one item at a time."

TIP

One item per submission

Staff submit the form once per product. If ordering 5 items, they submit 5 times. This keeps the spreadsheet clean and sortable by department or date.

TIP

Shorten the link

Use bit.ly or t.ly (free) to create a short link like bit.ly/resortorder — much easier to share on WhatsApp.

TIP

Add to phone home screen

On Android: open link in Chrome → three-dot menu → Add to Home Screen. On iPhone: open in Safari → Share → Add to Home Screen. It will look like an app icon.
